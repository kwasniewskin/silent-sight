### The Development Process
This project explores the capabilities of human-AI collaboration.
* **Concept & Design:** All gameplay mechanics, visual style, and game logic rules were designed by the developer.
* **Implementation:** The raw code (HTML/JS/Canvas) was generated by **Google Gemini** under strict supervision and iterative prompting.
* **Refinement:** The code was reviewed, debugged, and optimized by the developer to ensure gameplay balance and ethical constraints (e.g., target filtering logic).
